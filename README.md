## CigoAdminDemo
西谷后台CigoAdmin演示Demo

- 子牧
- zim@cigoos.com

## 主要特性

* 基于`Auth`验证的权限管理
    * 单管理员可配置多角色
    * 统一权限节点处理
    * 免登陆、免授权配置
* 方便快捷的前端组件
    * 完善的后台菜单及权限节点管理模块
    * CigoList插件：快速搭建后台数据检索列表
    * CigoEdit插件：快速进行表单开发，第三方富文本编辑器支持(Kindeditor、百度编辑器)
    * CigoUploader插件：快速进行文件上传的开发，多样化的上传示例，让你快速爱上它~~
* 通用的会员模块和API模块，Api接口支持统一加解密
* 支持二级域名，管理员及权限的分模块配置
